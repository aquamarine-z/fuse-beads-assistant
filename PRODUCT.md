# Product

## Register

product

## Users
Digital craft enthusiasts and makers who want to convert pixel art or photo images into physical Mard 221 fuse bead layouts, needing exact pixel grids and color code charts.

## Product Purpose
Provide an expressive, polished, and highly usable web interface to convert images into bead patterns. The tool enables live parameter adjustments, color quantization using the Mard 221 palette, grid chart generation with coordinate markers, color code legends, and large exports.

## Brand Personality
Expressive, polished, and highly precise. Integrates global theme/accent switching to feel premium and Material 3-like, while maintaining a clean tool-centric focus.

## Anti-references
Cluttered grids, low-contrast text on chart labels, arbitrary background decorations that distract from the pattern, and AI-typical visual slop (e.g. gradient text or excessive side-stripe borders).

## Design Principles
1. **Physical Fidelity**: The bead layout, color code coordinates, and legend map must represent physical fuse bead structures exactly.
2. **Context Preservation**: Seamlessly maintain user session state, active image data, and parameter selections across theme switches, language transitions, and page navigation.
3. **Legibility Under Load**: Ensure high contrast for coded chart letters/numbers and legend keys, even on small or dense grid layouts.

## Accessibility & Inclusion
Maintain WCAG AA compliance (contrast ratio >= 4.5:1) for text, code labels, and indicators. Support full light/dark mode adaptation, and respect system preferences for reduced motion.
